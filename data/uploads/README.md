# RAG Documents

Drop your documents here to be indexed for Retrieval-Augmented Generation (RAG).

## Supported formats
- `.md` — Markdown files
- `.pdf` — PDF documents
- `.txt` — Plain text files

## How it works
After adding files, trigger re-indexing via the app's **"Index Documents"** button (or restart the server).  
The FAISS vector index is stored in `data/vectordb/`.
